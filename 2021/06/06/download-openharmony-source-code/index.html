下载一份openharmony的源码

本文最后更新于：2022年8月22日 晚上

不知道为什么，总是有人告诉我鸿蒙已经开源了，不信可以自己去看源码balabala，其实鸿蒙的手机端目前为止依然没有开源，或者说没有完整完整开源。本文我将介绍如何拉取一份openharmony开源的源码。

首先需要准备以下东西

  • 一台装有类unix环境的电脑（wsl大概也行）
  • 6G磁盘剩余空间
  • 互联网（如果使用手机流量的话大概是1.5G）

  1. 使用repo正式开始同步源码
